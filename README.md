

## Install

1. cd $HOME &&```git clone https://github.com/luisadha/alrc-termux.git```

2. cd alrc-termux

3. make install

4. Manual step:
   source **~/.local/bin/alrc-termux.sh** from within your **~.bash\_profile** or **~/.bashrc** file
   
   Quick step:

   bash setup.bash.bashrc.conf

## Uninstall
   
   bash restore.bash.bashrc.conf
